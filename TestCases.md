# Тест-кейсы для проекта parabank_project

## TC-1: Успешный вход в систему

**Предусловия:**

- Пользователь с учетными данными:  
  **Username:** `john`  
  **Password:** `demo`  
  существует в системе.
- Доступна тестовая среда по базовому URL.

**Шаги:**

1. **Открыть базовый URL приложения:**  
   Запустить браузер и открыть URL тестовой среды.
2. **Ввести учетные данные:**
   - Ввести `john` в поле "Username".
   - Ввести `demo` в поле "Password".
3. **Нажать кнопку "Log In":**  
   Нажать на кнопку входа для отправки формы.
4. **Проверить отображение раздела "Accounts Overview":**  
   Убедиться, что после входа на странице присутствует текст "Accounts Overview".

**Ожидаемый результат:**

- После успешного входа отображается страница с текстом "Accounts Overview".

## TC-3: Успешный выход из системы

**Предусловия:**

- Пользователь успешно вошёл в систему (например, с учетными данными `john`/`demo`).

1. **Выполнить вход в систему:**  
   Использовать корректные учетные данные (например, `john`/`demo`).
2. **Найти и кликнуть по ссылке "Log Out":**  
   На странице после входа найти ссылку для выхода и нажать на неё.
3. **Проверить отображение страницы "Customer Login":**  
   Убедиться, что после выхода отображается страница входа с текстом "Customer Login".

**Ожидаемый результат:**

- После выхода пользователь перенаправляется на страницу входа, подтверждая, что сессия завершена.

## TC-7: Корректная оплата счета

**Описание:**  
Проверка корректной оплаты счета с использованием валидных данных.

**Предусловия:**

- Пользователь успешно вошёл в систему с учетными данными:  
  **Username:** `john`  
  **Password:** `demo`
- Доступна форма оплаты счета по URL `/billpay.htm`.

**Шаги:**

1. **Открыть базовый URL и выполнить вход:**
   - Запустить браузер и открыть URL тестовой среды.
   - Выполнить вход с использованием учетных данных `john`/`demo`.
2. **Перейти на страницу оплаты счета:**
   - Перейти по URL `/billpay.htm`.
3. **Заполнить форму оплаты счета:**
   - Ввести "Electric Company" в поле "Payee Name".
   - Ввести "456 Electric Ave" в поле "Address".
   - Ввести "City" в поле "City".
   - Ввести "State" в поле "State".
   - Ввести "67890" в поле "Zip Code".
   - Ввести "555-6789" в поле "Phone".
   - Ввести "987654" в поле "Account".
   - Ввести "987654" в поле "Verify Account".
   - Ввести "200" в поле "Amount".
   - Выбрать исходный счет (например, "13344") из выпадающего списка.
4. **Нажать кнопку "Send Payment":**  
   Отправить форму оплаты счета.
5. **Проверить отображение сообщения об успешной оплате:**  
   Убедиться, что на странице появляется сообщение "Bill Payment Complete".

**Ожидаемый результат:**

- После отправки формы отображается сообщение "Bill Payment Complete".
